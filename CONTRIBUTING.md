# Contributing to AI Learning Kit

First off, thank you for considering contributing to the AI Learning Kit! It's people like you that make this a valuable resource for the community.

## Guidelines for Contributions

To maintain the high quality and usefulness of this repository, please follow these guidelines when adding or suggesting resources:

### 1. Quality Over Quantity

- **Only target highest-quality resources.** There are thousands of AI materials available; we only want to list the best of the best.
- **Popularity matters.** Resources should be well-recognized in the industry or academia.
- **No half-baked or incomplete resources.** Ensure the content is complete, up-to-date, and provides real value to a learner.

### 2. Formatting & Icons

Maintain the existing aesthetic of the `README.md`. Please use consistent icons for different types of resources:

- 📍 for roadmaps.
- 🎥 for video-based courses/channels.
- 📘 for paid books or comprehensive guides.
- 📗 for free books or comprehensive guides.
- 📁 for code repositories.

### 3. Placement of New Resources

- **Default to the bottom:** New additions should generally go to the bottom of their respective category.
- **Exception for Excellence:** If a resource is undeniably "the absolute best" in its category, it can be placed higher up or at the top.

### 4. Categorization

- If you find a resource that doesn't fit existing categories, feel free to **suggest a new category**.
- Ensure the category title is clearly highlighted (and optionally include a relevant emoji).

## How to Contribute?

1. Fork the repository.
2. Create your feature branch (`git checkout -b contribution/AmazingResource`).
3. Commit your changes (`git commit -m 'Add AmazingResource'`).
4. Push to the branch (`git push origin contribution/AmazingResource`).
5. Open a Pull Request.

Thank you for helping us build the best AI learning kit!
